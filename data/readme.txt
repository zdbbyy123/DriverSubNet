The folder contains the data used in DriverSubNet method.
There are five subfolders BRCA, HNSC, KIRC and THCA which represent three different types of cancer,HPRD_Release9_062910 represents the HPRD release 9.
In each of subfolder, four datasets are listed.
Take the HNSC as the example, "mutation" means the processed mutation data integrated by single nucleotide variants and copy number variations.
"tumor_exp" represent the expression data of tumor samples.
"normal_exp" represent the expression data of normal samples.
"FGs.csv" represents the functional gene set.
"HPRD_network" represents the protein interaction network.
